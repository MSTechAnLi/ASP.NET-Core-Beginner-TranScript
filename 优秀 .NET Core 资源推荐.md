# 优秀 .NET Core 资源推荐

1. .NET Core 指南
    - 地址链接：https://docs.microsoft.com/zh-cn/dotnet/core/
    - 推荐理由：这是微软对 .NET Core 做出的官方指导，其重要性就不用我多言了，哈哈
